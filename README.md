# 🕳️ Holy Read - 现代圣经阅读应用

**Holy Read** 是一款专为现代读者设计的圣经阅读与学习应用。它结合了极简主义设计、先进的交互动画与强大的功能，旨在为您提供最纯净、最智能的灵修体验。

## ✨ 核心特性

- **多语种支持**：内置 **简体中文、繁体中文、英文** 版本，支持一键无缝切换界面与经文。
- **全局搜索**：全新的全局搜索功能，支持跨书卷实时检索经文内容，并伴有关键词高亮及快速跳转定位。
- **智能 TTS 朗读**：
  - **连续朗读模式**：朗读完当前节自动进入下一节，支持全书连读。
  - **自动翻页跟随**：朗读时页面自动滚动，确保当前内容始终处于视觉中心。
  - **速率调节**：支持 0.5x - 2.0x 朗读速度调节。
- **全屏阅读模式**：
  - 提供**仿真翻页、平滑滑動、垂直滾動**等多种分页模式。
  - 极简 UI 设计，消除干扰，专为深度阅读打造。
- **强大的学习工具**：
  - **范围高亮**：支持多种经文高亮颜色。
  - **范围收藏**：支持选择一段连续经文作为书签，支持本地搜索收藏内容。
  - **灵修笔记**：记录读经感悟，支持按内容实时搜索，并支持批量编辑与删除。
- **极致个性化**：
  - **主题系统**：明亮、深色、护眼三种基础模式。
  - **细节定制**：支持自定义背景色、主题強調色（Accent Color）。
  - **排版调节**：自由调整字体大小、行高及字体样式（宋体、楷体、圆体等）。

## 🚀 快速开始

### 运行项目

1. **安装依赖**：
   ```bash
   npm install
   ```

2. **启动开发服务器**：
   ```bash
   npm run dev
   ```

3. **构建生产版本**：
   ```bash
   npm run build
   ```

## 🛠️ 技术栈

- **前端框架**: React 18
- **构建工具**: Vite
- **语言**: TypeScript
- **设计系统**: Vanilla CSS (Fluid UI + Glassmorphism)
- **动画库**: Framer Motion
- **图标库**: Lucide React
- **持久化**: LocalStorage API
- **语音引擎**: Web Speech API
