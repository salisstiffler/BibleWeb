# 🕳️ Holy Read - 现代圣经阅读应用

**Holy Read** 是一款专为现代读者设计的圣经阅读与学习应用。它结合了极简主义设计与强大的功能，旨在为您提供最纯净、最智能的阅读体验。

## ✨ 核心特性

- **多版本支持**：内置简体中文、繁体中文及英文版本，支持一键无缝切换。
- **智能 TTS 朗读**：
  - 支持全章连读与单节点播。
  - **连续朗读模式**：开启后可自动阅读全书，支持自动翻页。
  - **自动滚动跟随**：朗读时页面自动滚动，确保当前内容始终处于视觉中心。
- **阅读记忆**：自动记录您最后的阅读位置，下次启动直达进度。
- **学习工具**：
  - **高亮标记**：支持四种颜色的划线功能。
  - **书签收藏**：一键收藏心仪经文，支持列表跳转。
  - **深度笔记**：随时随地记录读经感悟，支持按内容搜索、修改与删除。
- **个性化设置**：
  - **主题切换**：明亮、深色、护眼（Sepia）三种视觉模式。
  - **字体调节**：自由调整字体大小以适应阅读视力。
- **极致体验**：
  - 基于 React + TypeScript + Vite 构建，零延迟响应。
  - 采用流体动画与磨砂玻璃质感设计 (Glassmorphism)。

## 🚀 快速开始

### 运行项目

1. **安装依赖**：
   ```bash
   npm install
   ```

2. **启动开发服务器**：
   ```bash
   npm run dev
   ```

3. **构建生产版本**：
   ```bash
   npm run build
   ```

## 🛠️ 技术栈

- **前端框架**: React 18
- **构建工具**: Vite
- **语言**: TypeScript
- **设计系统**: Vanilla CSS (Fluid UI)
- **动画库**: Framer Motion
- **图标库**: Lucide React
- **持久化**: LocalStorage API
- **语音引擎**: Web Speech API
